HTML — The Structure

HTML provides the structural scaffolding. It describes what is on the page — headings, paragraphs, buttons, images — but has no knowledge of how it looks or behaves.

<button class="js-submit btn-primary">Submit</button>

Note: js- prefix = JS hook. btn-primary = CSS styling. Decoupled.

CSS — The Skin

CSS handles all visual states. When JavaScript toggles a class like is-active, CSS animates the transition. JS never touches inline styles directly.

.tab-panel  { display: none; }
.tab-panel.is-active { display: block; }

JavaScript — The Nervous System

JavaScript listens for events, processes state, and mutates the DOM by toggling CSS classes — never by writing inline styles. This is the professional standard.

const tabs = document.querySelectorAll('.js-tab');
tabs.forEach(btn => {
  btn.addEventListener('click', switchTab);
});

The DOM — The Living Map

The Document Object Model is the browser's live, in-memory representation of your HTML. It's rebuilt on every page load. JavaScript uses it as an API to read and update the page dynamically.

FAQ Accordion

JS toggles is-open. CSS animates max-height. Professional decoupling.

Input-Process-Output is the universal law of software engineering. Every interactive element follows it: an event fires (Input), JavaScript evaluates state and applies logic (Process), and the DOM updates visually (Output).

const signals intent — "this binding will never be reassigned." It prevents accidental mutations and makes code easier to reason about. Use let only when you know the value must change (like a counter). Never use var.

innerHTML parses strings as HTML, opening the door to Cross-Site Scripting (XSS) attacks if user input is injected. Always use textContent for user-provided data — it treats the value as plain text, never as markup.

js- classes are JavaScript hooks — never styled with CSS. is- classes represent visual state (e.g. is-active, is-hidden) — toggled by JS, styled by CSS. This decouples markup, style, and behaviour — the mark of professional code.
